Introduccion

Hello World o Hola mundo en español es una de las primeras cosas que debemos aprender a la hora de aprender un lenguaje de programación.

Los lenguajes de programacion mas populares en los que aprenderemos son:

  1. Python
  2. Java
  3. Javascript
  4. C
  5. Swift

Python

Python un lenguaje moderno y robusto muy interesante conocer, sirve para IA y seguridad informática aunque sus aplicaciones son infinitas.

# This program prints Hello, world! print('Hello, world!')

Las principales caracteristicas de este lenguaje son:

  • Lenguaje de programación de alto nivel
  • Es un lenguaje interpretado
  • Es multiparadigma
Java

Java uno de los lenguajes más usados en ámbitos móviles, por su interacción con el lenguaje Android y en la inteligencia artificial.

public class HolaMundo { public static void main(String[] args) { } }

Javascript

El lenguaje por defecto de front-end en la WEB JavaScript se gana su primer puesto como el lenguaje más usado y no es de extrañar.

<script> alert( 'Hello, world!' ); </script>

C

El viejo lenguaje C nunca pasa de moda y su inspiración a tantos lenguajes. Aquí su hola mundo.

#include int main() { // printf() displays the string inside quotation printf("Hello, World!"); return 0; }

Las caracteristicas principales de este lenguaje son las siguientes:

  • Es un lenguaje potente y eficiente, permitiendo obtener programas rápidos y compactos.
  • Proporciona un completo control de cuanto sucede en el interior del ordenador.
  • Permite una amplia libertad de organización del trabajo.
Swift

Swift el lenguaje por defecto en los productos de Apple, para programar en el iPhone

// Hello, World! Program import Swift print("Hello, World!")
